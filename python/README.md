### Python

#### Conda

Update conda

```
conda update -n base conda
```

Create virtual env

```
conda create -n <my env> <install module>
#  conda create --name snowflakes biopython
```

Check virtual env

```
conda info -e
```

Remove virtual env

```
conda remove -n <my env> --all
# conda remove -n snowflakes --all
```
